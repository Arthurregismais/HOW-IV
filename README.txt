Para que a aplicação da Locadora de Carros funcione perfeitamente, não se esqueça de baixar os seguintes arquivos sql : carro(1).sql, cliente.sql e usuario.sql
Nota: De o import dos arquivos sql na pasta mysql no localhost, senão será necessário realizar ligeiras alterações no código fonte 
Também abra o XAMPP control panel e aperte start nas opções Apache e MySQL
