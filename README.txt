Para que a aplicação da Locadora de Carros funcione perfeitamente, não se esqueça de baixar os arquivos sql que estam no repositório HOWIV


Também abra o XAMPP control panel e aperte start nas opções Apache e MySQL

os arquivos .sql que você baixar devem ser importados para o banco de dados mysql no localhost/phpmyadmin
